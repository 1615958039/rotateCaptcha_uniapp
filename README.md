### 运行项目
（请先安装并运行PHP接口端）
1. 下载hBuilderX(APP开发版)，安装node-sass插件
2. 克隆项目到本地，导入hbx
3. 打开项目目录,修改文件`./config.js`内相关配置
4. 打开hbx菜单栏找到`运行`->`运行到浏览器`->`Chrome`
